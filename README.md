# PyShop REST API

## About

This project was created for the REST API assignment.

The project is a simple e-commerce backend made using Python Flask. It allows users to add, view, update and delete products. A simple cart feature is also included.

The project uses SQLite to store the data.

## Technologies used

- Python
- Flask
- SQLite

## To Run the project

Install Flask first:

pip install Flask

Then run:

python app.py

The API will start on:

http://127.0.0.1:5000

## For Testing

Run:

python test_app.py
